# 서울시 공중화장실 데이터

## 원본

- 데이터셋: [공간화-서울시 공중화장실 정보](https://geomarket.kr/user/dataset/view.do?data_sn=1198)
- 제공기관: 한국국토정보공사 공간데이터마켓
- 유형: ZIP/XLSX 다운로드 데이터
- 배포주기: 1년
- 페이지 업데이트 날짜: 2026-04-10
- 구축범위: 서울특별시
- 명시 좌표계: `EPSG:5186 Korea 2000 / Central Belt 2010`

## 활용 판단

초기 화장실 시드 데이터와 데이터 모델 검증에 활용할 수 있다. 그러나 실시간 Open API가
아니고 관리자 승인 후 다운로드해야 하며 갱신주기가 1년이므로, 운영 서비스의 유일한
최신 정보원으로 사용하기에는 부족하다.

## 주요 원본 필드

| 원본 필드 | 의미 | 공통 모델 |
|---|---|---|
| `NUM` | 원본 순번 | `source_id` 후보 |
| `SE` | 개방·공중화장실 구분 | `details.category` |
| `TOILET_NM` | 화장실명 | `name` |
| `LCTN_ROAD_NM_ADDR` | 도로명주소 | `road_address` |
| `LCTN_LOTNO_ADDR` | 지번주소 | `address` |
| `OPN_TIME`, `OPN_TIME_DTL` | 개방시간 | `opening_hours` |
| `LAT`, `LON` | 위도·경도 | `latitude`, `longitude` |
| `DATA_WRT_YMD` | 데이터 작성일 | `source_updated_at` |

## 좌표계 주의

데이터셋 설명에는 EPSG:5186이 적혀 있지만 공개 샘플의 `LAT`, `LON` 값은
`37.x`, `127.x` 형태의 WGS84 위경도로 보인다. 실제 다운로드 파일에 투영좌표 컬럼이
별도로 있을 수 있으므로 다음을 확인한 뒤 사용한다.

1. `LAT`, `LON`을 카카오맵에 표시해 실제 위치와 비교한다.
2. EPSG:5186 좌표만 제공되는 파일이라면 WGS84 `EPSG:4326`으로 변환한다.
3. 주소와 좌표가 크게 어긋난 행은 제외하거나 검토 대상으로 표시한다.

## 다운로드 제약

- 이용 목적 선택과 보안각서 제출이 필요하다.
- 관리자 승인 후 다운로드할 수 있다.
- 다운로드 파일을 저장소에 그대로 커밋하지 않는다.
- 원본 사용 조건과 출처 표시 문구를 승인 후 다시 확인한다.

## 추천 수집 방식

```text
승인받은 원본 파일
→ 오프라인 인제스트 스크립트
→ 컬럼·좌표 검증
→ 공통 시설 모델 변환
→ 시설 저장소
```

운영 단계에서는 더 자주 갱신되는 공중화장실 Open API를 추가 조사한다. 이 데이터셋은
초기 구현과 비교 검증에 사용하고, 향후 다른 출처와 병합할 수 있도록 `source`와
`source_id`를 반드시 저장한다.
