# 마이 페이지

## 목적

사용자 계정과 앱 설정을 관리하게 한다.

## 주요 정보

- 로그인·로그아웃
- 프로필
- 언어 설정
- 위치 권한 안내

## 범위 판단

지도와 러닝 기록 흐름을 먼저 검증한다. 인증은 사용자별 기록 분리가 필요해지는 시점에 Supabase Auth로 추가한다.

## 현재 상태

- ID/PW 로그인 화면을 제공한다.
- ID/PW 회원가입 화면을 제공한다.
- ID와 비밀번호 입력 오류를 안내한다.
- Supabase Auth 클라이언트로 회원가입, 로그인, 세션 확인, 로그아웃 흐름을 연결한다.
- Supabase 환경변수가 없으면 실제 인증 요청 대신 설정 안내를 표시한다.
- 비밀번호 찾기, 이메일 인증, 소셜 로그인은 MVP 범위에서 제외한다는 안내를 보여준다.

## 완료 조건

- 로그인 상태와 로그아웃 상태가 명확히 구분된다.
- 위치 권한 문제를 해결할 수 있는 안내가 있다.
