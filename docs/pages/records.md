# 기록

## 목적

사용자가 이전 러닝 기록을 빠르게 탐색하고 상세 내용을 다시 보게 한다.

## 주요 정보

- 러닝 날짜
- 이동 거리
- 러닝 시간
- 메모 미리보기
- 기록 상세
- 메모가 있는 기록 필터
- 거리·시간·페이스 기준 정렬
- 월별 기록 필터
- 경로 미리보기

## 주요 상태

- 기록 로딩 중
- 첫 기록 없음
- 기록 목록
- 기록 조회 실패

## 범위 판단

초기에는 날짜순 목록, 상세 화면, 로컬 삭제를 구현했다. 이후 local-first 사용성을 높이기 위해 기록 탐색 기능을 확장한다. 복잡한 소셜 기능은 MVP에서 제외한다.

기록 탭은 local-first MVP에서 로그인 없이 열린다. 저장된 기록은 이 기기의 로컬 저장소 기준으로 조회한다.

메모가 있는 러닝만 모아보기는 마이 페이지 대시보드가 아니라 기록 페이지의 필터 기능으로 확장한다. 거리, 시간, 페이스 정렬은 기록이 쌓였을 때 “어떤 러닝이 좋았는지”를 빠르게 찾기 위한 탐색 기능이다.

## 완료 조건

- 저장한 기록이 최신순으로 보인다.
- 기록 상세에서 경로와 메모를 확인할 수 있다.
- 메모가 있는 기록만 모아볼 수 있다.
- 거리, 시간, 페이스 기준으로 정렬할 수 있다.
- 월별로 기록을 좁혀볼 수 있다.
- 저장한 기록을 삭제할 수 있다.
- 로그인하지 않아도 이 기기에 저장된 기록을 조회할 수 있다.

## 구현 기록

### 2026-06-21

- 하단 `기록` 탭에서 `localStorage`에 저장된 러닝 기록을 최신순으로 표시한다.
- 목록 카드에는 날짜, 거리, 시간, 평균 페이스를 표시한다.
- 상세 영역에는 거리, 시간, 평균 페이스, GPS 포인트 수, 메모를 표시한다.
- 상세 영역에서 기록 삭제 버튼을 제공한다.
- local-first 방향으로 전환하면서 기록 탭은 로그인 없이 바로 열리도록 변경했다.

### 2026-06-23

- 메모가 있는 러닝만 모아보는 필터를 추가했다.
- 월별 기록 필터를 추가했다.
- 최신순, 거리순, 시간순, 페이스순 정렬을 추가했다.
- 새로 저장하는 기록에는 `routePoints`를 함께 저장한다.
- 기록 상세에 경로 미리보기를 추가했다.
- 기존 기록처럼 경로 좌표가 없는 데이터는 “새 기록부터 경로를 볼 수 있다”는 안내를 보여준다.

### 2026-06-24

- 경로 미리보기 배경을 어두운 단색 SVG에서 밝은 지도풍 미니맵 스타일로 개선했다.
- 0km 또는 거의 이동하지 않은 기록은 직선 경로 대신 “이동 없이 머문 기록” 점으로 표시한다.
- 실제 지도 타일 위 경로 표시는 MapKit/지도 SDK 추가 렌더링 비용이 있어 추후 후보로 유지한다.

## 확장 후보

- 필터 UI 사용감 다듬기
- 기록이 없는 필터 조건의 empty state 개선
- 경로 미리보기 라벨과 시작·종료 표시 고도화
- 실제 지도 위 경로 표시
