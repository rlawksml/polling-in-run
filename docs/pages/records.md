# 기록

## 목적

사용자가 이전 러닝 기록을 빠르게 탐색하고 상세 내용을 다시 보게 한다.

## 주요 정보

- 러닝 날짜
- 이동 거리
- 러닝 시간
- 메모 미리보기
- 기록 상세

## 주요 상태

- 기록 로딩 중
- 첫 기록 없음
- 기록 목록
- 기록 조회 실패

## 범위 판단

초기에는 날짜순 목록과 상세 화면만 구현한다. 복잡한 통계 대시보드와 소셜 기능은 MVP에서 제외한다.

## 완료 조건

- 저장한 기록이 최신순으로 보인다.
- 기록 상세에서 경로와 메모를 확인할 수 있다.

## 구현 기록

### 2026-06-21

- 하단 `기록` 탭에서 `localStorage`에 저장된 러닝 기록을 최신순으로 표시한다.
- 목록 카드에는 날짜, 거리, 시간, 평균 페이스를 표시한다.
- 상세 영역에는 거리, 시간, 평균 페이스, GPS 포인트 수, 메모를 표시한다.
- 이동 경로 지도 시각화는 향후 기록 상세 고도화 단계로 분리했다.
