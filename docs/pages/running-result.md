# 러닝 결과

## 목적

완료한 러닝을 확인하고 개인 메모와 함께 저장하게 한다.

## 주요 정보

- 이동 거리
- 러닝 시간
- 평균 페이스
- 이동 경로
- 메모
- 사진

## 주요 상태

- 결과 확인
- 저장 전 메모 작성
- 저장 중
- 저장 완료
- 저장 실패

## 범위 판단

초기 MVP에서는 거리, 시간, 평균 페이스를 먼저 확인하고, 다음 단계에서 메모 저장을 연결한다. 사진 업로드와 상세 통계는 기록 저장 흐름이 안정된 이후 추가한다.

## 구현 기록

### 2026-06-21

- 러닝 종료 후 결과 확인 화면을 표시한다.
- 시간, 거리, 평균 페이스를 저장 전에 확인할 수 있다.
- GPS 포인트 수를 함께 안내해 계산 근거를 사용자에게 설명한다.
- 실제 기록 저장과 메모 입력은 다음 단계로 분리했다.

## 완료 조건

- 저장 전에 핵심 기록을 확인할 수 있다.
- GPS 포인트 기반 계산 상태를 이해할 수 있다.
- 저장 실패 시 작성한 메모가 사라지지 않는다.
