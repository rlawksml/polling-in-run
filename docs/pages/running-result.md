# 러닝 결과

## 목적

완료한 러닝을 확인하고 개인 메모와 함께 저장하게 한다.

## 주요 정보

- 이동 거리
- 러닝 시간
- 평균 페이스
- 이동 경로
- 메모
- 사진

## 주요 상태

- 결과 계산 중
- 저장 전 편집
- 저장 중
- 저장 완료
- 저장 실패

## 범위 판단

초기 MVP에서는 거리, 시간, 메모 저장을 우선한다. 사진 업로드와 상세 통계는 기록 저장 흐름이 안정된 이후 추가한다.

## 완료 조건

- 저장 전에 핵심 기록을 확인할 수 있다.
- 저장 실패 시 작성한 메모가 사라지지 않는다.
