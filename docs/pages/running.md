# 러닝 진행

## 목적

사용자가 달리는 동안 화면을 오래 읽지 않아도 현재 러닝 상태를 확인하고 제어하게 한다.

## 주요 정보

- 현재 위치와 이동 경로
- 경과 시간
- 이동 거리
- 일시정지
- 러닝 종료

## 주요 상태

- 러닝 진행 중
- 일시정지
- GPS 신호 정확도 낮음
- 위치 추적 실패
- 종료 확인

## UI 원칙

- 큰 숫자와 높은 대비를 사용한다.
- 조작 버튼은 최소화하고 충분히 크게 만든다.
- 실수로 러닝이 종료되지 않도록 종료 확인 단계를 둔다.

## 완료 조건

- 러닝 시작부터 종료까지 상태 전환이 명확하다.
- GPS 오류가 발생해도 이미 수집한 기록은 보존된다.
